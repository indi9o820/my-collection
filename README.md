# マイ・コレクション・マネージャー（ホビーコレクション管理アプリ）

このアプリは、散らかりがちなCD、DVD、ゲーム、書籍などの私的コレクションを、スマホからでも片手でサクサク登録・検索・管理できるデータベースツールです。「手軽に、爆速で、一生使える自分専用の図鑑」をコンセプトに、個人開発作品として開発しました。

## 成果物
- **公開URL**: https://my-collection-15a1e.web.app/

## 主な機能
- **インクリメンタルサーチ & 瞬時のカテゴリ絞り込み**:
　　入力した瞬間にリストが絞り込まれる爆速の検索機能と、ワンタップで切り替わるカテゴリフィルターを実装。目的のアイテムに一瞬でアクセスできます。
- **直感的なスマート・トグルソート**: 
　　セレクトボックスを廃止し、スマホでも片手で簡単に「新しい順」「古い順」を切り替えられる直感的なUI（トグルボタン）を採用しました。
- **データ負荷を抑える「もっと見る」ボタン**:
　　コレクションが数百件に増えてもブラウザの描画が重くならないよう、初期表示を20件に制限。slice を用いた動的な追加読み込み機能を搭載しています。
- **LocalStorage連携によるデータ永続化**:
　　データベースへの通信遅延を無くし、ローカル環境（ブラウザ）に安全にデータを保存。さらにデータの「インポート / エクスポート（JSON形式）」機能も備えており、機種変更時のバックアップにも対応しています。
- **ポートフォリオ用ダミーデータ内蔵**:
　　初回起動時に、著作権・商標クリーンな架空のダミーデータが22件自動で読み込まれるため、すぐに全ての機能（ソートや検索、もっと見るボタン）の動作確認が可能です。

## こだわったポイント・工夫した点
- **ユーザー体験（UX）を最優先した「引き算の設計」**:
  当初検討していた「あいうえお順ソート」は、日本語の漢字判定（フリガナ問題）によって直感的な並びにならないという課題に直面しました。そこで、あえて機能を「バッサリ削る」という判断をし、代わりに優秀な検索機能と1タップで切り替わる「追加順トグル」に絞ることで、迷いのない洗練された操作性を実現しました。
- **Reactのステート管理の厳格化とバグ防止**:
  データ更新（追加・編集・削除）時に発生しがちな「データの重複・増殖バグ」を徹底的に排除しました。Reactの関数型アップデート（prev => ...）を適切に使用し、IDの照合時には Number() による型合わせを行うことで、状態管理の堅牢性を高めています。
- **クリーンなポートフォリオ公開**:
  実在の製品名や企業名を使用せず、完全にオリジナルの架空データで初期構成を組んでいるため、商標トラブルのリスクが一切ない、ポートフォリオとして非常にクリーンな状態で公開しています。

## ローカルでの起動方法

```bash
# 1. リポジトリのクローン
git clone https://github.com/indi9o820/my-collection

# 2. プロジェクトフォルダへの移動
cd [クローンしたフォルダ名]

# 3. 依存関係のインストール
npm install

# 4. ローカルサーバーの起動
npm run dev
